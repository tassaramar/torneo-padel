# QA Prompts

Prompts operativos para testeo manual con agentes.

Convenciones:
- Los prompts de setup deben ser detallados, secuenciales y mecanicos.
- Los prompts de jugador deben definir identidad y objetivos, pero no explicar como cumplirlos.
- Si un prompt depende de precondiciones, esas precondiciones se preparan antes en un prompt separado.
